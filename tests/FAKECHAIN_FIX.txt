EXPLICAÇÃO DO ERRO
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PHP exige que valores padrão de propriedades de classe e parâmetros
de métodos sejam "expressões constantes" — ou seja, valores que o
compilador consiga avaliar em tempo de compilação, sem executar código.

Não permitido:
    public string $x = '0x' . str_repeat('f', 40);  // ← chamada de função
    public string $x = 'a' . 'b' . str_repeat(...);  // ← idem

Permitido:
    public string $x = '0xffffffffffffffffffffffffffffffffffffffff';  // literal
    public string $x = '0x' . 'ff';  // concatenação de literais
    public const  X  = '0xff';       // constante (pode usar em propriedades)
